# LLM-Plot

### LLM-Plotは簡単なCSV形式のデータをLLMを使って自動でプロットするソフトウェアです

![sample image](https://github.com/kasys1422/LLM-Plot/blob/main/img/screenshot.png?raw=true)

### 注意事項

ベータ版なのでバイナリは提供していません。自分でpython環境の構築を行う必要があります。
